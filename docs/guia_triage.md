# Guía del Analista de Triage y Gobernanza

Esta guía explica cómo utilizar el sistema de Triage de **GT Data Consulting** para transformar ideas en requerimientos técnicos estructurados y gobernados.

## 1. El Proceso de Vibe Coding
El sistema está diseñado para que los usuarios (Creadores o Autores) ingresen sus ideas en lenguaje natural. No es necesario preocuparse por el formato técnico inicial.

### Pasos:
1.  **Selecciona tu Sector**: Esto generará un ID de solicitud único (ej: ARQ-12345).
2.  **Define el Objetivo**: Describe brevemente qué se quiere lograr.
3.  **Describe la Idea**: Usa el campo "La Idea (Vibe Coding)" para escribir todo lo que tienes en mente.
4.  **Interactúa con la IA**: El panel de la derecha analizará tu texto y te hará preguntas para refinar la idea.

## 2. Gobernanza y Glosario de 3 Capas
Para asegurar la calidad de los datos, el sistema clasifica los términos técnicos en tres niveles:

-   **Capa GOBIERNO**: Términos intocables definidos por las políticas de la empresa.
-   **Capa TÉCNICO**: Definiciones técnicas sugeridas por los ejecutores o arquitectos.
-   **Capa OBTENIDO**: Términos detectados por la IA de fuentes externas que deben ser validados.

## 3. Auditoría de Calidad
El sistema realiza una auditoría en tiempo real mientras escribes. Si ves alertas en color rojo, significa que el requerimiento es ambiguo o le faltan datos críticos como el ROI.
